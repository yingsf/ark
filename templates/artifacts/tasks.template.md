## 状态定义

| 状态 | 含义 |
|------|------|
| Todo | 尚未开始 |
| Doing | 当前正在进行 |
| Done | 已完成且已验证（验证记录见 validation.md） |
| Blocked | 因依赖或外部条件无法继续，必须记录解除条件 |

## 任务格式

每条任务遵循以下结构：

```
- [ ] T{id} {任务描述}
  优先级：P0 / P1 / P2
  依赖：T{id}, T{id}（无依赖则省略）
  完成信号：什么情况下算这个任务完成（必填）
  当前批次：Batch n/m（{批次摘要}）（仅大型任务，无批次时省略）
  验证：validation.md #{章节} 或验证方式（Done 任务必填）
  阻塞原因：...（仅 Blocked 任务）
  解除条件：...（仅 Blocked 任务）
```
